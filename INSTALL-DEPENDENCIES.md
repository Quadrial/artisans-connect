# Required Dependencies Installation

## Backend Dependencies
Run these commands in the `backend` folder:

```bash
cd backend
npm install socket.io
```

## Frontend Dependencies
Run these commands in the root folder:

```bash
npm install socket.io-client
```

## After Installation
Restart both the backend server and frontend development server.

### Backend
```bash
cd backend
npm start
```

### Frontend
```bash
npm run dev
```

The messaging system will now work with real-time Socket.IO communication!
